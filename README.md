# Meow Meow TETRIS(=^・・^=)
Pygameで作った猫ミノのテトリス風ゲームです。

# ☆使い方☆
１．リポジトリをダウンロードします。
２．neko_tetris.py　を実行してください。

# ☆必要な環境☆
・Python　3.10以上
・pip install pygame が必要です。
・お好みでフォントを変えてみてください。
　デフォルトでは、pygameのSysFontが表示されます。

# 操作方法
・左右キー：移動
・下キー：落下を速める
・上キー：回転
・スペースキー：難易度選択画面で、猫を何度も降らせられます。
